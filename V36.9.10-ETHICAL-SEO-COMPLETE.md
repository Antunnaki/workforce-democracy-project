## ğŸ“ Technical Notes

### Canonical URL Format
All canonical URLs use absolute paths:
```html
<link rel="canonical" href="https://workforcedemocracyproject.org/page.html">
```

This is the recommended format by Google and prevents issues with:
- HTTP vs HTTPS confusion
- www vs non-www confusion
- Trailing slash inconsistencies
- Query parameter variations

### Image Requirements
For optimal social media sharing, ensure:
- **Dimensions**: 1200x630 pixels (Open Graph standard)
- **Format**: JPG or PNG (WebP not universally supported yet)
- **File Size**: < 300KB for fast loading
- **Alt Text**: Always included for accessibility
- **Self-Hosted**: On own server, not external CDN with tracking

**Current Image**: `https://workforcedemocracyproject.org/images/social-preview.jpg`

### Referrer Policy
We use `strict-origin-when-cross-origin`:
- âœ… Sends full URL to same-origin requests
- âœ… Only sends origin (not full URL) to cross-origin HTTPS
- âœ… Sends nothing to downgraded HTTP requests
- âœ… Privacy-preserving while maintaining functionality
