### Method 2: Use File Manager App (If project is local)

**For iPhone (Files app):**

1. **Open Files app**
2. **Navigate to project folder**
3. **Tap "Select"** (top right)
4. **Tap files to select** (blue checkmarks)
5. **Tap trash icon** to delete
6. **Confirm deletion**

**For Android (Files by Google or similar):**

1. **Open Files app**
2. **Navigate to project folder**
3. **Long-press a file** to start selection
4. **Tap more files** to select multiple
5. **Tap delete icon**
6. **Confirm deletion**

**TIP**: Sort by date - Delete everything before Jan 24, 2025 EXCEPT the 30 keep files
