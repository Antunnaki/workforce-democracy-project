## ğŸ” Vetting Process

### Step 1: Initial Research
1. Identify potential news source
2. Visit website and read "About Us" page
3. Check funding disclosure and ownership
4. Review corrections policy

### Step 2: Factual Accuracy Check
1. Look up source on [Media Bias/Fact Check](https://mediabiasfactcheck.com/)
2. Verify factual accuracy rating is "High" or "Very High"
3. Review any recent fact-checking issues
4. Cross-reference with other fact-checking organizations if needed

### Step 3: Content Review
1. Read 10-15 recent articles
2. Verify sources are cited clearly
3. Check for sensationalism, misleading headlines
4. Assess overall journalistic quality

### Step 4: Accessibility Check
1. Test website without account/subscription
2. Verify key news is accessible for free
3. Check RSS feed availability
4. Confirm no tracking/surveillance tech (privacy check)

### Step 5: Decision & Documentation
1. Add to vetted sources list with full metadata
2. Document bias rating (transparency)
3. Set priority level (1 = highest quality)
4. Schedule quarterly re-review
