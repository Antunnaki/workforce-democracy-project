# V36.0.7: Learn More Button Centering Fix

## Issue Reported
User reported that the "Learn More" button in the footer section was not centered on the homepage.

## Root Cause
The website has TWO "Learn More" buttons:
1. **Support Card Section** (lines 3296-3312): Before the footer, with gradient background
2. **Footer Section** (lines 3344-3352): Inside the actual footer

Both buttons were inside parent containers with `text-align: center`, but they lacked an explicit wrapper div to ensure proper centering on all devices, especially mobile.

## Solution Applied

### Fix 1: Support Card Section (Line 3298-3310)
**Changed:**
- Removed unnecessary nested `<div>` wrapper
- Added explicit centered wrapper div around the button

**Before:**
```html
<div style="text-align: center;">
    <div>
        <h3>üíù Support This Work</h3>
        <p>...</p>
        <a href="donate.html">Learn More</a>
    </div>
</div>
```

**After:**
```html
<div style="text-align: center;">
    <h3>üíù Support This Work</h3>
    <p>...</p>
    <div style="text-align: center;">
        <a href="donate.html">Learn More</a>
    </div>
</div>
```

### Fix 2: Footer Section (Line 3344-3352)
**Changed:**
- Wrapped the "Learn More" button in a centered div

**Before:**
```html
<div class="footer-section">
    <h3>üíù Support Our Work</h3>
    <p>...</p>
    <a href="donate.html" class="btn btn-secondary">Learn More</a>
</div>
```

**After:**
```html
<div class="footer-section">
    <h3>üíù Support Our Work</h3>
    <p>...</p>
    <div style="text-align: center;">
        <a href="donate.html" class="btn btn-secondary">Learn More</a>
    </div>
</div>
```

## Technical Details

### Why This Works
1. **Explicit Centering**: By wrapping buttons in `<div style="text-align: center;">`, we ensure proper centering across all devices
2. **Inline-Block Display**: The buttons maintain `display: inline-block`, allowing them to be centered within their parent
3. **No CSS Conflicts**: Using inline styles ensures no external CSS can override the centering behavior
