---

## Root Cause Found

**Old Approach (V32.7.4-7):** Hardcoded inline styles  
```html
<p style="color: rgba(255,255,255,0.98);">Text</p>
<div style="background: linear-gradient(...);">Section</div>
```

**New Approach (V32.7.8):** Design system variables  
```html
<div class="privacy-card">
    <p>Text</p>
</div>

<style>
.privacy-card {
    background: var(--surface);
    color: var(--text-primary);
}
</style>
```
