## If You Find Issues

### Invisible Text
- Check browser console for CSS loading errors
- Verify `unified-color-scheme.css` is loaded
- Hard refresh (Ctrl+Shift+R / Cmd+Shift+R)

### Layout Broken
- Check viewport width (resize browser)
- Check browser zoom level (should be 100%)
- Hard refresh to clear cache

### Buttons Not Working
- Check browser console for JavaScript errors
- Verify `js/security.js` and `js/personalization.js` are loaded
