## The Problem
Header text "Workforce Democracy Project" was **invisible** on learning page (white text on white background).

---

## The Root Cause
1. **CSS Selector Mismatch**: CSS targeted `header`, `.header`, `nav.header` but HTML uses `.site-header`
2. **Wrong Color Logic**: Text forced to white (`#FFFFFF`) for transparent header over dark hero
3. **No Hero on Other Pages**: Learning/FAQ/etc pages have no dark background behind header

**Result**: Header transparent + white text + white page background = INVISIBLE! ğŸ‘»

---

## The Solution (NO WORKAROUNDS!)

### Fix 1: Match the Selector
```css
/* Added .site-header to match actual HTML */
header,
.header,
.site-header,  /* â† FIXED! */
nav.header {
  background: rgba(255, 255, 255, 0.98);
}
```

### Fix 2: Use Dark Text on White Background
```css
/* Changed from white to dark grey */
.site-header .site-title {
  color: var(--text-primary);  /* Dark grey instead of white */
}
```

### Fix 3: Update Link Colors Too
```css
/* Added .site-header to all link selectors */
.site-header a { color: var(--text-primary); }
.site-header a:hover { color: var(--primary); }
.site-header a.active { color: var(--primary); }
```
