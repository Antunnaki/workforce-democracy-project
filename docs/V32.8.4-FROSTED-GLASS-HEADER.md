## ğŸ”® Advanced CSS Techniques Used

### 1. Backdrop Filter
```css
backdrop-filter: blur(20px) saturate(180%);
-webkit-backdrop-filter: blur(20px) saturate(180%);
```
**Effect**: Blurs and enhances colors of content behind element  
**Browser Support**: Modern browsers (95%+ coverage)  
**Fallback**: Slightly more opaque background for older browsers

### 2. RGBA Colors
```css
background: rgba(255, 255, 255, 0.75);
```
**Effect**: White background at 75% opacity  
**Benefit**: Allows underlying colors to show through  
**Precision**: Perfect control over transparency level

### 3. Text Shadow for Readability
```css
text-shadow: 0 1px 2px rgba(255, 255, 255, 0.8);
```
**Effect**: Subtle white glow behind purple text  
**Benefit**: Ensures readability on any background  
**Subtlety**: Not visible as "effect", just improves contrast

### 4. CSS Variables for Consistency
```css
color: var(--primary);
color: var(--primary-dark);
```
**Effect**: Uses defined brand colors  
**Benefit**: Change one variable, update entire site  
**Maintainability**: Single source of truth for colors
